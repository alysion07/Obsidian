# iPWR 시뮬레이터

- V-SMR 프로젝트에서 분석한 원자로 시뮬레이터
- 주요 화면 구성: overview, control, core, trips, systems
- Core 모니터링: liquid density, neutron flux, steam temp, void fraction
- 연료봉 상태: Boric Acid Addition, 농도 조절 (boration/dilution)
- Trips: Reactor trips/setbacks/stepback, Turbine trips/runback, ADS/PDHR actuation
- Systems: 원자로 전체 + 2차 계통 표시
- Rods Position Control: Reactor/Turbine 리딩 모드 선택, 수동/자동 전환
- Related: [[SMR]], [[MARS-GUI]]
