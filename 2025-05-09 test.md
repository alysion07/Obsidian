## 3. 트립(Trips)

### 변수 트립(Variable Trip)
- **Variable Trip**
  - Trip 401: SIS-SBLOCA (Always False for Steady-State)
  - Trip 402: FLB-SBLOCA (Always False for Steady-State)
  - Trip 430: Steam Line High Pressure
  - Trip 431: Steam Line Low Pressure
  - Trip 432: Feedwater Low Flow
  - Trip 500: Always True
  - Trip 501: Time > Trip 502 + 1.1
  - Trip 502: Reactor Trip (Reserved, Always False for Steady-State)
  - Trip 503: Turbine Trip (Always False)
  - Trip 506-509: Main Steam/Feedwater Isolation Valve Off (Time > Trip 731)
  - Trip 510: Always False
  - Trip 511-514: PRHR Isolation Valve On (Time > Trip 731)
  - Trip 570: Control Rod Insert (Trip 502 + 0.5 sec)
  - Trip 598-599: Reflood Related

### 논리 트립(Logical Trip)
- **Logical Trip**
  - Trip 603: RCP Velocity Control Trip
  - Trip 730: (Trip 430 OR Trip 431)
  - Trip 731: PRHRS Actuation Signal (Trip 730 OR Trip 432)
