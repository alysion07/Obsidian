## IA 구성 요소

IA에는 일반적으로 IA 구조의 기반이 되는 네 가지 주요 구성 요소가 있다.

1. **Organization systems** 정보를 분류하고 구성하는 방법
	 방/화장실/테라스가 몇 개 인지
	1. 계층 구조 (Hierarchical): 트리 구조, 가장 흔한 웹사이트 메뉴 구조.
	2. 순차 구조 (Sequential): 단계별 프로세스, 회원 가입이나 결제 과정.
	3. 매트릭스 구조 (Matrix): 사용자가 직접 기준을 선택하여 탐색하는 구조 (예: 쇼핑몰의 필터링).
		1. 알바벳 순 
		2. 연대별 순 
		3. 주제 순 
		4. 청중 순 
		
    
2. **Labeling systems** 정보를 표현하는 방법
	1. 화장실이기 때문에 변기가 있음
	2. 주방이기 때문에 싱크대가 있음
    
3. **Navigation systems** 사용자가 정보를 탐색하거나 이동하는 방법
	1. 현관 -> 주방 -> 침실 -> 테라스 순서대로 간다
    
4. **Searching systems** 사용자가 정보를 찾는 방법
	1. 검색 가능한 형태, 분류할 수 있도록 하는 방법
