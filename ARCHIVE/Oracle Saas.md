#### 특장점 
- 타사 대비 GPU 가격 최적화 가능
- 초기 산출된 견적에서 +-10% 견적 오차로 비교적 적은 편차를 제공함
	- 10TB 트래픽 무료처리 (쓱닷컴은 이걸로 충분했음)
- AWS 는 주로 MSA 아키텍쳐를 강조함 
	- 트러블슈팅에 많은 리소스 필요
	- HW적 한계로 인해 너무 마이크로서비스화 되어 있음
- 타사 대비 AI / ML 기능은 약한편
- Strorage가 가장큰 비용이 들어감 오라클에서는 1/4 가격으로 제공

- CPU, GPU, 메모리 사용량을 자유롭게 조정 가능함 (CPU만 100코어 가능)
- 비즈니스 크기가 증가함에 따라 비용 증가폭이 리니어함
- AMD CPU 사용시 조금 더 저렴
- ISMS 별도 취득 필요없음
	- ISMS-P는 별도 취득필요

- KINS 망 인프라 사용 가능 (사설망 구축or 레이턴시 최소화)
	- 비용은 매우 높음
###### A.I 서비스
- baremetal  여부에 따라 기준 나뉨 (1y ~3y 게런티시 할인 가능)
	 - 고비용: A100 / H100 등등 ( 직접 데이터 학습 )
	 - 학습 없이 추론에 적합한 장비 별도 제공
 - 그 이하는 VM 형태로 제공 On/Off 가능 & Compact 
 - 별도의 슈퍼클러스터링도 제공함
 
- O.S 라이선스 비용은 별도로 차징됨
#### 기존 CEA 솔루션의 개발환경
 - 보통 OS 라이선스 지불하고  기존 운영 환경을 유지하는 케이스가 대부분
 - 일반적으로는 고객별 어카운트 발급하여 소규모 운영환경을 배포
#### 과금방식 에 대한 질의응답
 - 아웃바운드되는 로우데이터에 대한 질문
	 1. 결과데이터 사이즈를 최소화하여 사용
	 2. 전용망을 사용한 정액제 사용 (유지보수 및 망설치 비용이 매우 높음)
	 3. VPN의 경우는 별도의 요금 발생 + 레이턴시 발생

GPU 리소스 사용에 대한 질문
	 1. NVLINK 지원 여부에 대한 답변은 추후 전달
	 2. 서버 클러스터링은 가능하나 사용 승인에 대한 절차 발생됨
	 3. 베어메탈을 사용하면 GPU 8장 까지 지원 가능함
	
망분리의 경우 VCN을 통해 논리적 격벽 생성 가능함
RDMA( remote direct memory access) 100gb 송수신 가능

- 사용량 및 task 모니터링을 위한 페이지
- Gen APP 형태로 NFLOW를 제공
