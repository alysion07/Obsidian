`#surrogate 

핵심 개념: **"원래의 것을 대신하거나 대표하는 존재"**

### 1. **언어학 (문자 인코딩에서)**

- 유니코드(UCS/UTF-16)에서 **서로게이트 코드(코드 유닛)**는 **1개의 문자를 표현하기 위해 2개의 코드 유닛으로 구성된 것**을 의미함
    
- 예: UTF-16에서 `U+1F600`은 하나의 글자지만, 이를 표현하려면 `\uD83D` + `\uDE00`의 두 개 코드 유닛이 필요
    
- 여기서 두 코드 유닛은 **서로게이트 쌍(surrogate pair)**라고 부르며, **문자를 대신해서 나타내는 코드**라는 점에서 '서로게이트'란 용어가 사용된다.

| 분야    | 의미                           |
| ----- | ---------------------------- |
| 일반    | 원래 것을 대신하는 대체물               |
| 언어학   | 특정 문자를 대신하는 코드 유닛 (UTF-16 등) |
| 사회    | 대리인, 대리모 등                   |
| 심리학   | 감정이나 욕망을 대신하는 행동             |
| DB/통계 | 직접 측정/식별 불가능한 것을 대신하는 키나 변수  |
