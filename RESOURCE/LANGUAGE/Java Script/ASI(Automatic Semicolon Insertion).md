### ASI 세미콜론 삽입 규칙

1. 줄바꿈이 되는 행의 마지막에
2. 행의 마지막이 `}` 이고, 줄바꿈이 되어 다음 행이 시작될 때
3. 파일의 끝에 도달할 때
4. `return`이 있는 행의 마지막에
5. `break`가 있는 행의 마지막에
6. `throw`가 있는 행의 마지막에
7. `continue`가 있는 행의 마지막에
