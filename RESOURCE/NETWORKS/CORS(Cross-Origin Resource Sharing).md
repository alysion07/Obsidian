# Origin이란 무엇?

서버의 위치를 의미하는 https://google.com 과 같은 URL들은 마치 하나의 문자열 같이 보여도, 사실은 여러개의 구성 요소로 이루어져 있다.

![[Pasted image 20240924085729.png]]
