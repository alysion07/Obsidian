## ETRI 진행사항

- ETRI에서는 [**Min IO**](https://min.io/)를 통해 스토리지 구현 중  
- AWS S3와 유사한 기능을 제공하는 오브젝트 스토리지 오픈소스 
- Log Data 전달 방식 고민 필요 
	- **SSE** 프로토콜 사용 예정이었으나 여러 제한 사항으로 인해 보류 
	- 함수를 호출하면 Task Manager에서 클라이언트로 **스트림 연결 하는 방식**으로 우선 사용 예정
- 로그인에 대한 부분 논의 필요

## NodeEditor 진행 사항 

- BPMN.js로 진행하려 하였으나 제공하는 기능이 개발 방향과 다른 부분이 있어 별도로 구현하는 방식으로 변경
- 현재 하나의 컴포넌트만 MARS Input 매뉴얼 기반으로 우선 적용 (Single Volume Component)
- 구현 기능
	- Component / Palette / Inspector 세 가지 패널로 편집 화면구성
	- 팔레트 -> 캔버스 드래그 & 드롭을 통한 컴포넌트 생성 및 편집 
	- 각 컴포넌트 별 프로퍼티 수정 
	- Undo / Redo 기능 (Create, Delete, Modify에 관한 History 지원)
	- SNGLVOL Component에 한하여 MARS Input 데이터 생성 
	- Input 미리보기
