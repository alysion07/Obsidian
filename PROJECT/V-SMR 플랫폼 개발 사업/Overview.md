#### 고속 해석 시뮬레이터 개발 TF 구성
자율 운전을 위해서는 실시간으로 해석에 사용되는 변수를 제어(값 변경)할 수 있어야 함. 그러나 현재 MARS의 경우 **실시간 해석 변수 제어 불가능한** 상태임.

MARS 코드 해석 흐름
- 해석에 필요한 각 카드를 작성 (Time 별 계통 컨디션 까지 미리 작성하여 입력함)
- 해석 시작 시 모든 데이터 입력 (한번 해석이 진행되면 별도의 조작 어려움)
- 결과를 모니터링 하는 방식

#### 결론
MARS 코드 내부의 제어 가능한 변수가 있는지, 수정 확장 가능한 부분이 있는지 검토가 필요함
