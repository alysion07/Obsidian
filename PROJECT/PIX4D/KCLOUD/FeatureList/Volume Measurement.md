**부피 측정 동작 흐름** 
1. 사이드바 -> Measurement 영역 -> 부피 측정 기능 선택 
2. 메인 뷰 내의 특정 지점을 클릭하여 Polygon 형태의 영역 지정 
3. 영역 선택 완료 시, 트리에 볼륨 측정 아이템 생성 
4. 생성된 아이템 클릭시 Property 창에 관련 정보 팝업 
5. 부피 측정 버튼을 클릭하여 측정을 시작함 
6. 측정 완료 시 화면에 렌더링 (생성된 삼각형  + 높이 값에 따른 삼각형의 색상)

**이슈 사항**
5. 부피 계산 시, 비동기 or 계산 최적화로 화면 프리징 개선 필요
	- 계산 성능 최적화 필요( n^2 -> nlogn  )
6. 들로네 삼각 분할로 필터링 된 PCD로 생성되는 삼각형 정보 최적화 필요
	- 삼각형 개별 -> 통합 렌더링 
	- 복잡한 형상일 때, 예외 처리: 인접한 점과 이루는 각이 둔각인 경우 초기 설정한 영역과 형상이 다르게 삼각형이 생성되는 이슈 
	- 높이 값 색상 그라데이션 적용: 색상값을 자연스럽게 렌더링하기 위해 별도의 셰이더 생성 필요
