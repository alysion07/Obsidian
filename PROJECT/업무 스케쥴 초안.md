### 1. 신규 인력 담당 업무 기술 방향 수정 필요

신규 인력의 업무가 시뮬레이션 결과 시각화 요구사항 분석으로 기술되어 있으나, 현재 수행 중인 핵심 업무는 시뮬레이션 입력 모델링 GUI 개발에 집중하고 있습니다. 
입력 모델링 GUI 설계 및 개발로 변경하되, 향후 다중 물리해석 모델 지원 및 해석 결과 분석 기능 확장 예정임을 포함하여 기술하는게 어떨까 합니다. 

### 2. 2차년도 핵심 과업 범위 조정 필요

고속 스트림 데이터 분산 메시징 구조 설계, 컨테이너 기반 물리모델 커널 형상관리 등은 세부과제 전체 수준의 과업으로, ETRI가 주관하는 영역입니다. 원자로 계통 구성요소 시각적 편집 기능 고도화, GUI-시뮬레이터 연계 인터페이스 개발 등으로 조정하는게 어떨까합니다.

### 3. 기존 인력 업무와의 일관성 확보 필요

기존 인력은 "GUI 인터페이스 기획 및 개발"로 기술되어 있으나, 신규 인력은 "결과 시각화"로 방향이 상이하게 기술되어 있음. 기존 역할의 승계·확장 맥락에 맞게 일관성 있는 기술을 권고함.
